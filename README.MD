### Deskripsi Program
Aplikasi ini adalah katalog produk hardware sederhana yang dibangun menggunakan framework Django.
Fitur utama:
- Menampilkan halaman utama (Home).
- Menampilkan daftar produk yang diambil dari data hardcoded di views.
- Menampilkan detail informasi setiap produk berdasarkan ID.
- Halaman kontak informasi.

### Cara Menjalankan
1. Install Django (`pip install django`).
2. Jalankan perintah `python manage.py runserver`.
3. Akses `http://127.0.0.1:8000/` di browser.
