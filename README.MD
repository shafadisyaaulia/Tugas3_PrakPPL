### Deskripsi Program:
Aplikasi ini adalah katalog hardware berbasis Django yang menampilkan daftar produk secara dinamis tanpa database. Menggunakan Template Inheritance untuk efisiensi kode dan Bootstrap 5 untuk tampilan yang modern.

### Cara Menjalankan:
1. Pastikan Django terinstall (`pip install django`).
2. Masuk ke folder project.
3. Jalankan `python manage.py runserver`.
4. Buka `http://127.0.0.1:8000/` di browser.
